# User Readme

## Project layout

- Place your ROS 2 packages in `src/`.

## Run a dev container

1) Pick the container you want to use.
   Example: `tigerVNC/04_jazzy_tigerVNC_ros2-control`
2) Open that folder in VS Code.
3) Use "Dev Containers: Reopen in Container".

## Open the GUI

- Browse to `http://localhost:6080`
- Password: `ubuntu` (verify the ovewrite value in the dev-cnotainer json.)
